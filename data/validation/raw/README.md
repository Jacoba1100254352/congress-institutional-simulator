# Raw Validation Inputs

Curated, documented empirical extracts live in this directory. Small public API
adapter fixtures belong in `data/validation/fixtures/` and are intentionally
ignored by the raw validation-readiness check.

Current raw sample:

- `bill_progression.csv`: 180-row Congress.gov 118th Congress offset sample,
  generated by `scripts/validation/build_bill_progression_dataset.py`.
- `bill_progression.metadata.md`: generation parameters and sample-level floor,
  committee-report, and enactment rates.

The current sample is useful for empirical-bridge checks and rough
agenda-access plausibility. It is not a complete congressional census and should
not be described as fitted empirical validation.
